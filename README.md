# Flutter Smart Habit & Productivity Tracker App

## 📌 Project Description
This is a Flutter-based Smart Habit and Productivity Tracker App.
It helps users track daily habits, monitor progress, and improve productivity.

## 🚀 Features
- Add new habits
- Track daily completion
- View progress analytics
- User-friendly UI
- Local storage support

## 🛠️ Technologies Used
- Flutter
- Dart
- Firebase (if used)
- SQLite / Local Storage

## 📂 Project Structure
lib/
 ├── models/
 ├── screens/
 ├── widgets/
 ├── services/
 └── main.dart

## ▶️ How to Run
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
